# Huzurlu Bahçe

**Huzurlu Bahçe** is an interactive web application designed to support cognitive and motor skills exercises. The app offers various games and activities aimed at helping users practice memory, attention, orientation, coordination, and drawing skills in a fun and engaging way.

## Features

- Cognitive exercises such as memory games, attention tasks, and orientation activities
- Motor skill exercises including shape drawing and hand-eye coordination tasks
- Progress tracking and achievements display
- Simple and intuitive user interface with engaging visuals and icons

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser.
3. Navigate through the menus to select and play different exercises.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Baloo 2)
- FontAwesome Icons
